# 风险控制规则

## 集中度风险

1. 黄金占比超过30%：高集中度风险，暂停新增黄金。
2. 任一行业主题基金占比超过10%：暂停新增该主题。
3. 单一高波动主题基金占比超过8%：提示波动风险。
4. 宽基核心仓低于15%：提示组合底盘不足。

## 数据风险

1. 数据质量为低时，不输出明确买卖建议。
2. 主动基金估值置信度必须标低。
3. 债基不做精确盘中估值。
4. 开源数据源异常时，必须使用手动快照或跳过操作建议。

## 行为风险

1. 当日查看估值次数过多，应提示减少盯盘。
2. 当日连续多次触发买入冲动，应提示等待周报。
3. 单日大涨后不追高。
4. 单日大跌后也不允许突破最大单日买入金额。

## 流动性与费用风险

1. 换基前检查赎回费。
2. 检查基金是否有持有期限制。
3. A/C类选择必须结合预期持有时间。
4. 债基滚动持有期未到，不建议切换。

## 合规边界

1. Skill不提供基金销售服务。
2. Skill不自动执行交易。
3. Skill不构成投资建议。
4. 所有估值和评分仅作为个人辅助参考。
