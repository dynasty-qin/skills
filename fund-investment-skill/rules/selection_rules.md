# 基金选择与评分规则

## 总原则

1. 基金筛选必须同类比较。
2. 近1个月、近3个月收益只能作为观察指标，不能作为买入或换基唯一依据。
3. 收益必须结合回撤、波动、基金经理、费率、规模、风格稳定性一起看。
4. 指数基金和主动基金使用不同评分体系。
5. 当前仓位偏离目标时，优先调整仓位，不优先换基金。

## 被动指数基金评分

适用：ETF联接、指数LOF、普通被动指数基金。

| 指标 | 权重 |
|---|---:|
| 跟踪误差/跟踪偏离 | 30% |
| 费率成本 | 20% |
| 基金规模 | 20% |
| 成立时间 | 10% |
| 基金公司指数能力 | 10% |
| 流动性/申赎便利 | 10% |

判断要点：

- 这类基金的目标是稳定跟踪指数，不是显著跑赢指数。
- 近一年收益差异不应被过度放大。
- 规模过小、跟踪误差异常、费率明显偏高，应降低评分。

## 指数增强基金评分

适用：中证A500指数增强、沪深300增强等。

| 指标 | 权重 |
|---|---:|
| 相对基准超额收益 | 25% |
| 超额收益稳定性 | 25% |
| 最大回撤控制 | 20% |
| 跟踪误差合理性 | 15% |
| 规模与费率 | 10% |
| 管理团队稳定性 | 5% |

判断要点：

- 近一年表现好只能进入观察，不可直接替代核心被动仓。
- 如果增强基金回撤显著高于基准，应降低评分。
- 如果超额收益只集中在短期，评分不得过高。
- A500核心仓可采用“被动为主、增强为辅”的内部结构。

## 主动权益基金评分

适用：主动医疗、消费、科技、均衡权益基金。

| 指标 | 权重 |
|---|---:|
| 基金经理任职期表现 | 20% |
| 3年/5年同类风险调整收益 | 20% |
| 最大回撤控制 | 20% |
| 风格稳定性 | 15% |
| 基金规模 | 10% |
| 持仓质量 | 10% |
| 费率与公告风险 | 5% |

判断要点：

- 基金经理变更后，历史业绩参考价值下降。
- 主动基金必须关注风格漂移。
- 亏损不等于基金差，要区分行业下跌和基金跑输同类。
- 同类长期跑输、回撤更大、风格漂移时，才进入观察。

## 债券基金评分

| 指标 | 权重 |
|---|---:|
| 最大回撤控制 | 30% |
| 收益稳定性 | 25% |
| 信用风险 | 15% |
| 久期风险 | 10% |
| 基金规模与持有人结构 | 10% |
| 费率 | 5% |
| 流动性 | 5% |

判断要点：

- 债基不追求最高收益，优先稳健。
- 高收益债基可能承担更高信用风险、久期风险或可转债风险。
- 有滚动持有期的债基不能当现金替代。

## 黄金基金评分

| 指标 | 权重 |
|---|---:|
| 跟踪黄金价格偏离 | 35% |
| 费率成本 | 20% |
| 基金规模 | 20% |
| 目标ETF流动性 | 15% |
| 申赎便利 | 10% |

判断要点：

- 黄金基金重点看跟踪和成本。
- 黄金类别主要问题通常是仓位控制，不是频繁换基金。

## A类/C类选择规则

1. 预计持有时间小于1年，优先考虑C类。
2. 预计持有时间大于1年，优先考虑A类。
3. 当前组合仍在调整期，优先C类。
4. 已确定为长期核心仓，可优先A类。
5. 换基前必须考虑申购费、赎回费、销售服务费和持有期限制。
