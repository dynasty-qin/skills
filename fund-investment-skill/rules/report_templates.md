# 报告模板规范

## 日报结构

```markdown
# 基金组合日报 - YYYY-MM-DD

## 一、今日结论

## 二、数据质量

## 三、组合概况

## 四、今日估算

## 五、仓位偏离

## 六、规则触发情况

## 七、今日建议

## 八、风险提醒
```

## 周报结构

```markdown
# 基金组合周报 - YYYY-WXX

## 一、本周结论

## 二、组合表现

## 三、盈亏贡献

## 四、仓位偏离

## 五、下周新增资金分配建议

## 六、风险提醒
```

## 月报结构

```markdown
# 月度基金优选报告 - YYYY-MM

## 一、本月结论

## 二、当前持有基金评分

## 三、候选基金池评分

## 四、观察名单

## 五、淘汰名单

## 六、是否建议替换

## 七、下月观察重点
```

## 季报结构

```markdown
# 季度策略复盘 - YYYY-QX

## 一、季度结论

## 二、资产配置是否仍合适

## 六、下一季度策略
```
